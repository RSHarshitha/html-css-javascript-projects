# Rotating Navigation

This project demonstrates a rotating navigation UI with animated transitions. The main content features information about Ferrari, including a brand image and description. The navigation menu rotates in and out, providing quick access to Home, About, and Contact sections.

## Features
- Rotating navigation animation
- Ferrari brand info and image
- Responsive design
- Font Awesome icons for navigation

## How to Use
1. Click the menu button to open or close the navigation.
2. Explore the Ferrari information and image in the main content area.
3. Use the navigation links to switch between sections.

## Technologies
- HTML
- CSS
- JavaScript
- Font Awesome
