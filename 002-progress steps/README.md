# Progress Steps

This project demonstrates a progress steps UI component with star-shaped steps. Each step is labeled (A-D) and visually highlights progress as you move forward or backward using the navigation buttons.

## Features
- Star-shaped progress steps
- Active step highlighted
- Responsive design
- Easy navigation with Previous/Next buttons

## How to Use
1. Click "Next" to advance to the next step.
2. Click "Previous" to go back to the previous step.
3. The progress bar and step highlight update automatically.

## Technologies
- HTML
- CSS (with `clip-path` for star shapes)
- JavaScript
